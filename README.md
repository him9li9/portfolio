# Nastya Portfolio (Next.js)

Portfolio landing + case study page implemented with:

- Next.js (App Router)
- Tailwind CSS
- Framer Motion

## Local run

1. Install Node.js 20+
2. Install dependencies:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Import repo in Vercel.
3. Framework preset: `Next.js` (auto-detected).
4. Build command: `npm run build`
5. Output: `.next`

No extra env vars required.

## Routes

- `/` - main portfolio page
- `/app` - AceCamp case study page
